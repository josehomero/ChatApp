<h1>Chat App</h1>

<br>
A chat app where you can make an account and share your thoughts or start
a conversation with your friends.


##Summary
A chat applications that uses ReactJS for the front end and C# for the core and
back end. Sass is used for styling. PostgreSQL is used for saving the messages.
